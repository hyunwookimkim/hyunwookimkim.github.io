HTMLCollection

HTMLCollection은 테그들을 모아놓은 것이다.
HTMLCollection 객체는 배열은 아니지만,
배열처럼 length 프로퍼티가 있고,
인덱스 번호를 이용하여 찾은 태그들을 꺼낼 수 있다.
=> 그래서 반복문을 사용할 수 있다.

querySelector

querySelector는 Element()의 프로토타입이다.
querySelector는 "CSS selector 문법"을 쓴다.
=>getElementsByClassName/id/class 이런거 보다 querySelector("") 를 추천한다. 왜냐하면 CSS selector 문법을 사용해서이다.

1) 아이디로 찾기
2) 태그 이름으로 찾기
3) 분류명으로 찾기
4) CSS selector 문법을 사용하기 때문에 복잡한 조건을 기술할 수 있다.
=> 조건에 해당하는 태그가 여러 개일지라도 그 중 첫 번째 것만 리턴한다.
=> .g2.g3는 g2와 g3지만, .g2 .g3는 g2속의 g3이다
5) 조건에 해당하는 태그를 찾지 못했다면?
=> 리턴 값은 null이다.

querySelectorAll

querySelectorAll는 "CSS selector 문법"을 쓴다.
=> NodeList 객체 리턴
NodeList는 배열이 아니다. 따라서 다음 반복문을 사용하면 필요없는 프로퍼티까지 반복하게 된다.
NodeList가 배열이 아니지만, 배열처럼 length 라는 변수가 있고,
인덱스를 통해 항목을 꺼낼 수 있다. 따라서 다음과 같이 반복문을 사용할 수 있다.

1) 아이디로 찾기
2) 태그 이름으로 찾기
3) 분류명으로 찾기
4) CSS selector 문법을 사용하기 때문에 복잡한 조건을 기술할 수 있다.
5) 조건에 해당하는 태그를 찾을 수 없다면?
=> 빈 NodeList 객체이다.

querySelectorAll과 getElementsByTagName 사용시 비교

getElementsByTagName는 *HTMLCollection로 리턴하는데, 리스트 삭제등 변경시 실시간 반영이 되어서 리스트 유지가 굳이 필요없으면 사용하면 된다.
querySelectorAll는 *NodeList로 값을 리턴하는데, 변경사항이 있어도 리스트를 유지하고 싶으면 사용하면된다.

해석하기
("article:first-child > p");//부모의 첫번째 자식인 아티클을 찾는다.그리고 직계 자식인 p태그를 찾는다.
("article:nth-child(2) > p");//부모의 두번째 자식인 아티클을 찾는다.그리고 직계 자식인 p태그를 찾는다.
("article:last-child > p");//부모의 두번째 마지막인 아티클을 찾는다.그리고 직계 자식인 p태그를 찾는다.
e.parentElement.removeChild(e);// e를 삭제하고 싶으면 부모엘리먼트로 가서 리무브 차일드로 e를 삭제해줘라.

.innerHTML, .textContent, .innerText사용시 비교

Element.innerHTML : DOMString
=> Element() 생성자가 추가한 프로퍼티이다.
=> 태그의 콘텐트를 HTML *코드* 그대로 리턴한다.

Node.textContent
=> Node() 생성자가 추가한 프로퍼티이다.
=> 태그의 콘텐트에서 HTML 코드를 제거한 후 *텍스트*만 리턴한다.

HTMLElement.innerText
=> HTMLElement() 생성자가 추가한 프로퍼티이다.
=> 태그의 콘텐트에서 HTML 코드를 제거한 후 *텍스트*만 리턴한다.

.setAttribute사용

=> 원래 태그에 있던 속성 값을 바꿀 때
태그객체.setAttribute("속성명", "값")
=> 이 메서드를 사용하면 개발자가 임의로 추가한 프로퍼티 값도 바꿀 수 있다.
즉 원래 태그 객체에 없던 프로퍼티 값도 설정할 수 있다.
e.setAttribute("href", "http://www.google.com");

appendChild()는 자식노드(태그)를 맨 뒤에 붙인다.

appendChild(text); 노드객체를 붙힐때는 appendchild를 넣어야 한다.
p.appendChild(text2); //h1.innerHTML 로 텍스트만 붙히면 사용가능하지만,
노드에 붙힐땐 appendchild를 써야 한다.
